# 将下列代码单独拷贝到 main.cpp（全部替换）中进行独立功能测试

## 1. test_ENCODER.cpp - 编码器测试

### 测试步骤：
1. 连接电机 A/B 的霍尔编码器 A/B 相到对应引脚（见 UserConfig.h）
2. 上传代码到开发板
3. 打开串口监视器（115200 波特率）
4. 手动旋转电机输出轴
5. 观察角度和速度输出
6. 检查两个编码器方向是否一致

### 预期结果：
- 串口输出连续的圈数和速度值
- 旋转方向应与输出变化方向一致
- 静止时速度值应接近 0
- 两个编码器应独立工作，互不干扰

## 2. test_MOTOR.cpp - 直流电机测试

### 测试步骤：
1. 连接 TB6612 电机驱动到开发板（PWM/IN1/IN2）
2. 上传代码到开发板
3. 打开串口监视器（115200 波特率）
4. 观察电机 A/B 是否依次正转、刹车、反转、滑行

### 预期结果：
- 两个电机都能正转和反转
- 刹车时电机迅速停转
- 滑行时电机自由减速
- 无异常噪音或抖动

## 3. test_MPU6050.c - MPU6050 传感器测试

### 测试步骤：
1. 连接 MPU6050 到 I2C 总线 0（引脚 8-SDA, 9-SCL）
2. 上传代码到开发板
3. 保持传感器静止进行校准
4. 打开串口监视器（115200 波特率）
5. 观察初始校准信息
6. 移动传感器观察数据变化
7. 测试不同方向的加速度和旋转

### 预期结果：
- 校准过程应顺利完成
- 静止时加速度 Z 轴应接近 1g（重力）
- 旋转时陀螺仪数据应有相应变化
- 温度数据应在合理范围内
- 数据输出稳定无跳变
